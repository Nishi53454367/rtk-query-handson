# ローカル環境構築手順
## ビルド
```
docker-compose build
```

## node_modulesインストール
```
docker-compose run --rm front_node sh
yarn install
exit
```

## コンテナ起動
```
docker-compose up -d
```

## 画面起動
http://localhost:3000
